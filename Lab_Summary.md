This lab implemented a RAG pipeline using OpenAI embeddings, cosine similarity retrieval, metadata filtering, LLM-based relevance scoring, and reranking. Metadata filtering successfully restricted retrieval to either the Trustworthy AI transcript or the Ethics Guidelines PDF. Reranking was most useful when the initial retrieval results were already relevant but not optimally ordered, allowing more relevant chunks to be promoted. I would recommend reranking in applications where retrieval precision is important, such as legal, compliance, or policy-focused systems, although the baseline embedding retrieval performed well on this dataset.
